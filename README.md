# Modern Calculator — Electron (JavaScript) Version

This project is a port of the original Python calculator into an Electron-based desktop app using HTML/CSS/JavaScript.

## How to run (developer)
1. Install Node.js (14+).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the app:
   ```bash
   npm start
   ```

## Features
- Modern dark UI
- Basic & advanced math operations
- Memory functions (MC, MR, M+, M-)
- Keyboard support
- Safe-ish expression evaluation (no remote code execution)

## Files
- `main.js` — Electron main process
- `preload.js` — preload script
- `index.html`, `styles.css`, `renderer.js` — frontend
- `package.json` — metadata and start script
